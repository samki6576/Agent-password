## 🚀 How to Use

### Option 1: Run Locally (5 minutes)

```bash
pnpm install
cp .env.local.example .env.local
# Edit .env.local with your Auth0 credentials
pnpm dev
# Open http://localhost:3000
```

**Then:**
- Click "Connect Google Calendar"
- Follow Auth0 OAuth flow
- See your calendar events
- Run `python ai-agent.py` to test gateway

### Option 2: Deploy to Vercel (15 minutes)

1. Push code to GitHub
2. Connect GitHub repo to Vercel
3. Add 4 environment variables in Vercel Dashboard
4. Update Auth0 callback URLs
5. Test at your Vercel domain

See `DEPLOYMENT.md` for detailed steps.

### Option 3: Submit to Hackathon (30 minutes)

1. Deploy to Vercel (from Option 2)
2. Record optional demo video using `DEMO_SCRIPT.md`
3. Use text from `HACKATHON_SUBMISSION.md`
4. Submit with:
   - GitHub repo link
   - Vercel deployment URL
   - Demo video (optional)
